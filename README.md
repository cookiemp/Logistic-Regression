# Logistic Regression from Scratch

## Course: Introduction to Artificial Intelligence  

## Institution: Addis Ababa University  



## Description

This repository contains a manual implementation of Logistic Regression for binary classification using Python and NumPy. It avoids high-level ML libraries to demonstrate the underlying mathematics of the Sigmoid activation function and Cross-Entropy Loss.

1. Install dependencies:

```bash

pip install numpy matplotlib
```
